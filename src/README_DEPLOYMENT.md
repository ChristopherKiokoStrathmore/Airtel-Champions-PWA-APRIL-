## 🆘 SUPPORT & TROUBLESHOOTING

### **Common Issues:**

1. **"Gradle build failed"**
   - Solution: `cd android && ./gradlew clean`

2. **"App is white screen"**
   - Solution: Check `chrome://inspect` → Console

3. **"Plugin not implemented"**
   - Solution: `npx cap sync android`

4. **"Emulator is slow"**
   - Solution: Increase RAM to 4GB in AVD settings

### **Debugging Tools:**
- 🔍 Chrome DevTools: `chrome://inspect`
- 📋 Android Logcat: Android Studio
- 🩺 Capacitor Doctor: `npx cap doctor`
