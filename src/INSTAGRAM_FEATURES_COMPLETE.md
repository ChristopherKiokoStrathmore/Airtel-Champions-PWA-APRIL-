## 💬 COMMENTING FEATURE

### **How It Works:**

```
1. Open any post (tap in feed or grid)
   ↓
2. Detail modal opens
   ↓
3. Scroll to bottom
   ↓
4. See input: "Add a comment..."
   ↓
5. Type your comment
   ↓
6. Press Enter OR tap [Post] button
   ↓
7. Comment appears immediately!
   ↓
8. Other users see it too
```

**Features:**
- ✅ Real-time comment posting
- ✅ Director comments show 👑
- ✅ Your name appears automatically
- ✅ Timestamp shows "Just now"
- ✅ Comments saved to database

## ❤️ LIKING FEATURE

### **How It Works:**

**In Feed View:**
```
Tap ❤️ icon → Heart turns RED → Count increases
Tap again → Heart turns gray → Count decreases
```

**In Grid View:**
```
Hover → See like count
Click photo → Detail modal → Tap ❤️ at bottom
```

**In Detail Modal:**
```
❤️ 💬 📤 buttons at bottom
Tap ❤️ → Animates RED → "48 likes"
```

**Features:**
- ✅ RED hearts (Airtel brand!)
- ✅ Heart pop animation
- ✅ Instant feedback (optimistic UI)
- ✅ Can unlike by tapping again
- ✅ Your likes saved to database

## 🔄 USER FLOWS

### **Flow 1: Upload Photo Post**

```
Home → Tap Feed icon (📸)
     → Tap [+ New]
     → Tap large photo area
     → Select photo
     → Photo appears (square preview)
     → Write caption (optional)
     → Check consent box
     → Tap [🚀 Post]
     → Success!
     → Post appears in feed
```

### **Flow 2: View in Grid**

```
Feed screen → Tap Grid icon (⊞)
           → See 3x3 photo grid
           → Hover over photo
           → See likes & comments overlay
           → Tap photo
           → Detail modal opens
           → See full photo + all comments
```

### **Flow 3: Like & Comment**

```
Grid view → Tap photo
          → Detail modal opens
          → Tap ❤️ at bottom
          → Heart turns RED
          → "48 likes" updates
          → Scroll to comment input
          → Type "Great work!"
          → Press Enter
          → Comment appears with your name
          → Close modal
          → Return to grid
```
