# How to Add TAI to Your Phone's Home Screen

## 🦅 Make TAI Look Like a Real App!

Follow these simple steps to install TAI on your phone so it appears next to WhatsApp, Instagram, and your other apps.

---

## 📱 For iPhone/iPad (iOS)

### Step 1: Open Safari
- **Important:** Must use Safari browser (not Chrome)
- Go to your TAI app URL

### Step 2: Tap the Share Button
- Look for the share icon at the bottom (box with arrow pointing up)
- Tap it

### Step 3: Add to Home Screen
- Scroll down and find "Add to Home Screen"
- Tap it

### Step 4: Confirm
- You'll see the TAI eagle logo preview
- Name will show as "TAI"
- Tap "Add" in the top right

### ✅ Done!
- Go to your home screen
- **TAI eagle icon is now there!** 🦅
- Tap it to open TAI like a native app
- No browser bars, full screen experience!

---

## 📱 For Android Phones

### Method 1: Chrome Prompt (Easiest)
1. Open TAI in Chrome browser
2. Chrome might show a banner saying "Add TAI to Home screen"
3. Tap "Add"
4. **Done!** TAI eagle icon appears on home screen 🦅

### Method 2: Manual Install
1. Open TAI in Chrome browser
2. Tap the menu (3 dots in top right)
3. Select "Add to Home screen" or "Install app"
4. A dialog appears with TAI eagle logo preview
5. Tap "Add" or "Install"
6. **Done!** TAI eagle icon appears on home screen 🦅

## 🔧 Troubleshooting

### "I don't see Add to Home Screen option"
- **iOS:** Make sure you're using Safari (not Chrome)
- **Android:** Make sure you're using Chrome (not other browsers)
- Clear browser cache and try again

### "The icon looks blurry"
- This is normal with the current setup
- For crisp icons, see `/APP_ICON_SETUP.md` for optional icon optimization

### "It opens in the browser still"
- After adding to home screen, tap the **new icon** on your home screen
- Don't use the browser bookmark
