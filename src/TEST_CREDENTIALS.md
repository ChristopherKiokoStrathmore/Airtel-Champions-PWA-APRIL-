## 🔧 **SETUP INSTRUCTIONS**

### **Step 1: Create Auth Users in Supabase**

```bash
1. Open Supabase Dashboard
2. Go to "Authentication" → "Users"
3. Click "Add User" (manually)
4. For each test user above:
   
   Example (John Kamau):
   ├─ Email: john.kamau@airtel.co.ke
   ├─ Password: JohnTAI@2024!
   ├─ Auto Confirm User: ✓ (checked)
   ├─ User Metadata: 
   │  {
   │    "phone": "0712345001",
   │    "full_name": "John Kamau"
   │  }
   └─ Click "Create User"

5. **IMPORTANT:** Copy the UUID that's generated
   Example: a1b2c3d4-e5f6-7890-abcd-ef1234567890

6. Repeat for at least 20 users:
   - Top 10 Field Agents (for leaderboard testing)
   - 5 ZSMs (one per zone)
   - 5 ZBMs, HQ, Director (management testing)
```

### **Step 2: Update seed-data.sql**

```sql
-- Replace placeholder UUIDs with real UUIDs from Step 1

-- BEFORE (in seed-data.sql):
('11111111-1111-1111-1111-111111111111', 'john.kamau@airtel.co.ke', ...)

-- AFTER (with real UUID from Supabase):
('a1b2c3d4-e5f6-7890-abcd-ef1234567890', 'john.kamau@airtel.co.ke', ...)

-- Use Find & Replace in your SQL editor
```

### **Step 3: Run seed-data.sql**

